# echarts-for-weixin

此目录需要放置 echarts 小程序组件文件。

## 获取方式

从 https://github.com/ecomfe/echarts-for-weixin 下载，将以下文件复制到此目录：

- `ec-canvas.js`
- `ec-canvas.json`
- `ec-canvas.wxml`
- `ec-canvas.wxss`
- `echarts.min.js`（从 echarts 官网定制下载，选择 K线图 + 柱状图 + 折线图）

## 定制 echarts

到 https://echarts.apache.org/zh/builder.html 定制下载，至少选择：
- 折线图
- 柱状图
- K线图
- 直角坐标系
- dataZoom
